Auto Popup Expiry is a simple Auto Deactive Popup for Wordpress

== Description ==

Don't forget to remove the popup!
The main feature of APE PopUP is the auto expiration, will be enough to select the publication end date and the popup will be automatically removed.
You can choose between Classic Popup and Notification Popup, both allow total customization, let's see them below:

Classic PopUp:
• Automatic deactivate after certain date
• Customize Width and Height (px / auto)
• Set Automatic "FadeOut" Timeout
• Upload or Select image by WP Media
• Choice the background-color of the Popup
• Set the border-radius value
• Insert a custom link (external or internal)
• Simple Shortcode [Ape] to display the Popup

Notification PopUp:
• Customize Content, <HTML> Friendly
• Automatic deactivate after certain date
• Customize Width and Height (px / auto)
• Set Automatic "FadeOut" Timeout
• Choice the background-color
• Choice the font-color
• Set the border-radius value
• Set the indoor margin value
• Customize position (Top, Left, Right, Bottom)
• Custom Animation IN
• Custom Animation OUT
• Simple Shortcode [Ape] to display the Popup

== Installation ==

1. Upload the entire `ape-auto-popup-expiry` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Auto Popup Expiry' menu in your WordPress admin panel.
